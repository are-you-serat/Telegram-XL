Telegram XL - это форк Telegram X с функциями нарушающими TOS. 


## Фичи клиента:
1) Удаление спонсорской рекламы
2) Обход защиты скриншотов в секретных чатах
3) Игнорирование таймера самоуничтожения

## В разработке:
1) Ghost mode
2) Антиудаление сообщений
